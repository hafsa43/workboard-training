# Day 3: React Router - Navigation and Protected Routes

## What I Built Today
- Login page for user authentication
- Dashboard page as the main landing area
- Projects list page showing all projects
- Project detail page (utilizing URL parameters)
- Custom 404 error page
- Protected route wrapper component
- Active navigation styling for current routes

## Key Learnings
- **React Router**: Comparable to NavigationManager in Blazor
- **Routes**: Similar to @page directive in Blazor
- **useNavigate**: Hook for programmatic navigation
- **useParams**: Extracts URL parameters (like [Parameter] attribute from routes)
- **useLocation**: Retrieves current route information
- **Link**: Similar to NavLink component in Blazor
- **Navigate component**: Used for redirecting users
